# night-swarm
《夜幕虫潮：Night Swarm》是一款基于 HTML5 Canvas 开发的俯视角生存动作小游戏。玩家扮演废城中的守夜人，在不断涌来的异变虫群中移动、自动攻击、收集经验并选择升级。游戏操作简单，使用 WASD 或方向键控制移动，攻击由系统自动完成，玩家重点需要判断走位路线、躲避包围，并在升级时选择更适合当前局势的强化方向。  游戏节奏紧凑，敌人会随着时间推移变得更多、更快，生存压力逐步增强。玩家可以通过提升伤害、攻速、移速、生命值和拾取范围来构建不同成长路线。当前版本已实现敌人生成、自动索敌、碰撞检测、经验掉落、升级选择、生命值、分数统计和重新开始等核心机制。项目无需后端，适合部署到 GitHub Pages，打开浏览器即可游玩，也便于后续扩展武器、Boss、角色和排行榜系统
